# Reaper Scripts for Game Audio

This is a collection of scripts that I wrote for Reaper. Feel free to use them and share them with the game audio community.
Have fun guys!

Sws, RealmGui, and js_ReaScriptApI extensions need to be installed for the scripts to run.

# Installation

You can install these scripts using ReaPack by navigating to: Extensions > ReaPack > Import Repositories
Copy and Paste this link https://raw.githubusercontent.com/Afrodiaq/Afrodiaq-Reascripts/refs/heads/master/index.xml
